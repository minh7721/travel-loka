Bánh mì

Nhìn vào vùng khoanh đỏ bạn thấy services này chưa được Start và nó đang ở chế độ Manual (tức là không tự động khởi động (Automatic)) Bây giờ chỉ cần click chuột phải vào dòng đó rồi chọn Start là bạn có thể login vào được SQL rồi. Để máy chủ SQL Server tự động khởi động cùng windows mà không cần phải vào đây để Start lại thì bạn thì Click phải chuột vào dòng đó, chọn Properties tại phần Startup Type: Chọn Automatic, đồng thời nhấn Start rồi nhấn OK để hoàn thành. Bây giờ thì bạn đăng nhập SQL bình thường nhé.
